# @DeinAdmin/deinadmin.codes
This is my personal website based on ReactJS with a little help of @material-ui/core.

## First steps:
1. If you wanna use it make sure to install the modules:
````
$ yarn

$ npm i
````
2. Run it:
````
$ yarn start

$ npm start
````
3. Building it for production:
````
$ yarn build

$ npm build
````
